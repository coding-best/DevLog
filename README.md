# 提交信息规范
> 建议使用英文编写提交信息，并在提交信息前添加具体信息类型。
- `feat`: 新功能
- `fix`: 修补 bug
- `refactor` 重构代码
- `style`: 调整代码格式
- `config` 修改配置文件
- `test`: 测试相关
- `chore`: 辅助工具的变动
- `doc`: 更新文档（README 等）
- `sql`: 数据库脚本相关
